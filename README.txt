Custom product badge for woocommerce.

== Description ==

Now, you can easily add product badge in woocommerce shop. Just add active the plugin then it will add custom sale badge. 

## Features: ##

**Sale Badge** Automatically add a sale badge in on sale product.

== Installation ==

= Modern Way: =
1. Go to the WordPress Dashboard "Add New Plugin" section.
2. Search For "Custom Product Badge for WooCommerce". 
3. Install, then Activate it.

= Old Way: =
1. Upload `Custom Product Badge for WooCommerce` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
